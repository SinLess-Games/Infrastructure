# Cloud Platforms Documentation and Resources

## What This Page Is For

This page is a curated reference for major cloud platforms and cloud-adjacent providers.

It is meant to make it easier to find the right official documentation quickly when working on:

- infrastructure
- Kubernetes
- platform engineering
- CI/CD
- networking
- storage
- observability
- identity
- APIs
- automation

5. Linode API

3. Hetzner
4. Cloudflare

## Notes

A few practical habits make cloud documentation more useful:

- prefer official docs first
- use architecture centers when designing systems
- use CLI and API references when automating
- verify region, pricing, and service availability before planning around a product
- check Kubernetes, networking, IAM, and observability docs together because those topics overlap heavily in real platforms
